# CSS Fundamentals

Welcome to CSS Fundamentals. This course takes you from zero to building real layouts with confidence.

## What You'll Learn

- How the CSS box model controls element sizing and spacing
- Flexbox for one-dimensional layouts (navbars, card grids, centering)
- CSS Grid for two-dimensional layouts (dashboards, magazine layouts)
- Responsive design patterns that work on every screen
- Modern CSS architecture with custom properties and cascade layers

## Prerequisites

- Basic HTML knowledge (tags, attributes, document structure)
- A code editor (VS Code recommended)
- No CSS experience required

## Course Structure

- **4 lessons**, ~30 minutes each
- Each lesson ends with interactive exercises
- Final capstone: build a landing page from a design

## Start Learning

Begin with **Lesson 1: The Box Model** — it's the foundation everything else builds on.
